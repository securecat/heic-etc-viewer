Run a local server to enable folder drag & drop.

In Command Prompt, navigate to the folder containing heic-etc-viewer.html and run:

With Node.js
npx serve .
With Python
python -m http.server 8080

Then open http://localhost:3000/heic-etc-viewer.html in your browser.
