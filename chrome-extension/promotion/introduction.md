"HeV Sender" collects the images, videos, and PDFs on the page you're viewing and opens them all at once in "HEIC etc Viewer (HeV)", a browser-based media viewer and converter.

Whatever HeV can display — including HEIC/HEIF photos that browsers normally can't preview — can be viewed and converted together in one place. See GitHub for the full list of supported formats and other details.

Source code and changelog are available on GitHub:
https://github.com/securecat/heic-etc-viewer/tree/main/chrome-extension

---

「HeV Sender」は、閲覧中のページにある画像・動画・PDFをまとめて、ブラウザで動くメディアビューアー＆コンバーター「HEIC etc Viewer (HeV)」で開くChrome拡張です。

ブラウザが通常プレビューできないHEIC/HEIF写真も含め、HeVが表示できるものを一か所でまとめて閲覧・変換できます。対応形式の一覧やその他の詳細はGitHubをご覧ください。

ソースコードおよび更新履歴はGitHubで公開しています：
