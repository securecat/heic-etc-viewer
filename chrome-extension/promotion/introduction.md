"HeV Sender" is a Chrome extension that sends images, videos, and PDFs from the page you're viewing to "HEIC etc Viewer (HeV)" all at once.

The supported file formats for sending are the same as those supported by HeV: HEIC/HEIF, JPEG, PNG, WebP, GIF, AVIF, SVG, BMP, TIFF, ICO, MP4, WebM, MOV, and PDF (as of HeV v3.15.0). If HeV adds support for additional formats in the future, HeV Sender’s supported formats will be updated accordingly.

Source code and changelog are available on GitHub:
https://github.com/securecat/heic-etc-viewer/tree/main/chrome-extension

「HeV Sender」は、閲覧ページ中の画像・動画・PDFをまとめて「HEIC etc Viewer (HeV)」へ送出するChrome拡張です。

送出対象となるファイル形式は、HeVの対応形式と同じ：HEIC/HEIF、JPEG、PNG、WebP、GIF、AVIF、SVG、BMP、TIFF、ICO、MP4、WebM、MOV、PDF です（HeV v3.15.0現在）。今後、HeVの対応形式が増えた場合は、HeV Senderの送出対象も追従します。

ソースコードおよび更新履歴はGitHubで公開しています：
https://github.com/securecat/heic-etc-viewer/tree/main/chrome-extension
